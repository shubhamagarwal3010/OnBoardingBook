The Books example app shows how to use the cloud recognition service to build a simple UI to scan a sample target target cover and display info on that target.

Key Functionality

  • Display reco query status
  • Request target info meta data based on reco response
  • Render target info on target
  • Transition display of target info to screen when off target

Targets

Sample Books

Instructions

  • Point camera at sample target cover to view info
  • Press close box to scan another target
